
>>> python mssql_scanner.py -h 192.168.43.42 -p 1433 -u users.txt -pw passwords.txt --safe



-h  目标主机的IP地址或主机名，可以不写，默认为 127.0.0.1

-p  要扫描的端口，可以不写，默认为 1433

-u  指定单个用户名或写有多个用户名的文件，可以不写，默认为“sa”；文件格式为每行一个用户名，不允许空格或其他无关符号，

-pw 指定一个密码或写有多个密码的文件，可以不写，默认为“admin”，文件格式同上

--safe 指定两次连接请求间的时间隔，可以不写，默认为0，单位为秒，“--safe”不写时间时，间隔为0.1秒
